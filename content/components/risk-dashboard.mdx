## Leadership Risk Dashboard

A comprehensive risk management dashboard featuring an interactive heat map and top risk drivers overview. Perfect for executive dashboards, security monitoring, and risk assessment applications.

<ComponentCodePreview name='riskdashboardcomponent' />

## Features

- **Interactive Risk Heat Map**: 3x3 grid visualization showing risk distribution by likelihood and impact
- **Hover Tooltips**: Detailed information on hover for each risk hotspot
- **Risk Drivers List**: Key risks requiring leadership oversight
- **Responsive Design**: Fully responsive layout with Tailwind CSS
- **Modern UI**: Dark theme with gradient backgrounds and smooth animations
- **Icon Integration**: Uses Lucide React icons for visual indicators

## Use Cases

- Executive dashboards
- Security operations centers (SOC)
- Risk management platforms
- Compliance monitoring
- Board presentations
- CISO reporting tools
